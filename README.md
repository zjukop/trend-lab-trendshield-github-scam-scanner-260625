# TrendShield — Scam & Malware Risk Scanner for GitHub Repos

GitHub Trending is increasingly polluted by fake crypto tools, keygens, and keyword-stuffed malware bait. Build a lightweight ‘TrendShield’ scanner that gives every repo a transparent scam-risk score and a shareable report. It rides the current wave of suspicious trending repos while being clearly defensive, useful, and star-worthy.

## Why This Repo

This project is generated by a trend-driven pipeline. It targets **Developers, security researchers, open-source maintainers, GitHub Trending users, and organizations doing lightweight repo due diligence** and focuses on the niche: **Open-source supply-chain safety for GitHub trend watchers**.

## Key Features

- CLI + GitHub Action that scans repos, READMEs, releases, and topics for scam/malware/credential-theft signals
- Heuristic rules for viral scam patterns: fake crypto balance/flash-USDT, seed phrase tools, cracked software/keygens, suspicious binaries, keyword stuffing, zero-issue star spikes
- Generates a shareable Markdown trust report with risk score, evidence snippets, and recommended safe actions
- Optional browser-bookmarklet or simple web UI where users paste a GitHub URL and get an instant report
- Maintainer-friendly allowlist/suppressions file so legitimate security research repos can document intent
